The machine learning in this assignment was done with RL_Sim.
RL_Sim may be obtained with the following link:
https://www.cs.cmu.edu/~awm/rlsim/

The mazes used in the report are found in the mazes folder.

For the mazes called small_snake_pneg05 and big_spiral_pneg05 run the rl_sim_for_0_5.jar

For the rest of the mazes run rl_sim.jar.
